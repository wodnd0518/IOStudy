# 이름 짓기 규칙
### **Lower Camel Case**

- function, method, variable, constant    
    > someVariableName
    

### **Upper Camel Case**

- type(class, struct, enum, extension…)   
    > Person, Point, Week
    

***Swift는 모든 대소문자를 구분합니다.***

## **출력하기**

- print 함수
    > 단순 문자열 출력
    
- dump 함수
    > 인스턴스의 자세한 설명(description 프로퍼티)까지 출력
   

# 상수 & 변수
### 상수(변하지 않는 일정한 값)

 let 상수명: 데이터타입 = 값

`let a: Int = 100`

a값은 200,300을 넣든 100으로 일정!!

### 변수(변할 수 있는 값)

var 변수명: 데이터타입 = 값
